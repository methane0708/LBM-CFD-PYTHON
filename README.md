# LBM-CFD-PYTHON
這是一個CFD二維模擬程式，使用LBM加上部分LES架構以模擬渦流以下的流體情況。


流場大小:6公尺*1.5公尺


時間尺度:5萬步=1秒






可調整變因:


上下牆壁(不穩定)


運算步驟數


輸入物品(自行輸入正方形圖片或內建橢圓及naca對稱翼形


攻角


流體速度


流體種類(4種)


模擬名稱




執行注意事項:


所有檔案皆須下載


運行程式請以專用終端執行cfd.py


程式在高雷諾數或高流體速度容易崩潰，故注意輸入條件


本程式使用pytorch，所以可以以gpu計算，但要裝cuda




執行流程:


使用gui輸入模擬條件


程式運行時會有進度條及顯示計算速度


完成計算會將結果轉換成vti以供後續至paraview觀看結果
